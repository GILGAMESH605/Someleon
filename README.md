# Someleon
Social_Chameleon_chat_agent

The Context-Aware Social Strategist.

Concept: An AI Agent that doesn't just "reply," but constructs psychological profiles of the interlocutor and browses the live web (via Firecrawl) to find niche communication strategies (e.g., specific sales closers, dating trends, or cultural references) before formulating a response.

打破人情障碍，让人人学会’潜规则‘的话术沟通agent
根据使用者的目的，实时分析最合适发的信息
无论是谈生意，想要追求对方，又或者是有诱导性的目的,对于追求性消息使用大量已有的‘海王’'海后‘的聊天记录，对于谈生意就用大量成功的生意人的聊天记录去培训ai，不断更新语料库，做到真正的‘接地气，’而不是ai的尴尬语气；实时通过网络泡虫在博客以及各种平台搜集补充语料，分析对方语言习惯以及大体性格，给出大方向交流方案以及下一步可供参考的信息；发送后可以进一步根据我方实际发送的信息以及对方回复的信息来继续分析下一步策略。对于聊天记录不仅可以上传文字，还可以上传图片等形式的聊天记录，由附带的工具自动转为文档，方便用户使用；下一步计划尝试接入iMessage等信息平台去更加优化体验，并且搜集大量现有的语料库加快简单任务的反应速度。

Someleon (Social Cameleon) is a multi-turn chat AI agent demoed in the browser. Users provide any communication objective (e.g., business negotiation, relationship repair, dating conversations) and a chat thread (text or screenshots). The agent analyzes the counterpart’s communication habits and likely needs using evidence from the transcript, produces a north-star strategy and a next-step plan, and generates multiple ready-to-send message options. After the user commits what they actually sent and the counterpart’s reply, the agent keeps updating the profile and strategy within the same session to drive multi-turn progress. Optional Firecrawl web search can be enabled to pull public communication templates/patterns, and the next milestone is integrating platforms like iMessage to improve input UX and speed.
